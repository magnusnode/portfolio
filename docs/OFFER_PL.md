# Oferta Łukasza Sojki 2026/27

## AI i IT - 50% oferty

1. Audyt workflow AI.
2. Sprint AI Operations.
3. Personal AI OS.
4. GitHub Repository Steward.
5. Obsidian / Notion Knowledge OS.
6. Automatyzacja n8n / Python / PowerShell.
7. Strona WWW / portfolio / landing page.
8. Windows / workstation / IT Ops.

## Cyber Intelligence i OSINT - 25% oferty

1. Audyt ekspozycji cyfrowej.
2. Scam i podszywanie - triage.
3. OSINT firmy / konkurencji.
4. Raport intelligence po incydencie.

Zakres jest defensywny, oparty na legalnych źródłach publicznych i materiałach przekazanych przez klienta. Brak nieautoryzowanego dostępu, śledzenia, doxxingu, credential access i działań ofensywnych.

## Inne usługi online - 25% oferty

1. Dokumentacja techniczna i SOP.
2. System produktu cyfrowego.
3. Research i briefing decyzyjny.
4. Knowledge Base Rescue.

## Model współpracy

- kontakt pisany jako domyślny;
- ankieta / brief zamiast obowiązkowego calla;
- fixed scope i fixed price tam, gdzie to możliwe;
- jedna pisemna runda poprawek w standardowym pakiecie;
- rozmowa tylko przy niepewnym zakresie lub integracjach;
- rezultat: plik, raport, strona, workflow, vault, PR lub ZIP.
